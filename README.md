# Análise de Ocupação Hospitalar - Araranguá (2022)

Este projeto consiste em uma análise exploratória e tratamento de dados sobre a ocupação de leitos hospitalares no município de Araranguá, Santa Catarina, durante o ano de 2022. O trabalho foi desenvolvido como projeto final para a disciplina PyAnalytics - Resolução de Problemas de Análise de Dados e Compartilhamento de Conhecimentos, oferecida pela UFSC.

## Contexto e Objetivos
O objetivo deste projeto foi aplicar os conhecimentos de análise de dados com Python adquiridos durante o curso para investigar padrões de ocupação em leitos de UTI e clínicos destinados ao tratamento de COVID-19. A análise foca na extração de insights sobre a pressão do sistema de saúde regional a partir de bases de dados públicas.

## Requisitos Técnicos Atendidos
O desenvolvimento seguiu os critérios estabelecidos para a conclusão do curso:
1. Seleção de dataset relacionado à saúde com dados regionais de bases públicas.
2. Realização de etapas de carga de dados utilizando bibliotecas específicas.
3. Análise exploratória para verificação de tipos de dados, colunas e dados faltantes.
4. Limpeza de dados e tratamento de inconsistências.
5. Operações com a biblioteca Pandas (agrupamentos, somas e transformações).
6. Visualização de dados por meio de bibliotecas gráficas.
7. Apresentação de insights e padrões obtidos.

## Tecnologias Utilizadas
* Python
* Pandas
* NumPy
* Matplotlib

## Metodologia Aplicada
A base de dados original foi filtrada para o município de Araranguá. Durante o tratamento, as datas foram convertidas para o formato adequado e valores nulos foram tratados como zero para garantir a precisão estatística nos cálculos de média mensal e total de óbitos hospitalares confirmados.

## Principais Insights
A análise demonstrou que o município apresentou picos de ocupação nos primeiros meses de 2022, seguidos por uma tendência de queda acentuada no segundo semestre. Os dados confirmam a eficácia das medidas de controle e vacinação no alívio do sistema hospitalar regional.
